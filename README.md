# KeyWrit Hub

A web application for generating JWT license tokens. Companion tool to the [KeyWrit](https://github.com/keywrit/keywrit) validation library.

## Usage

Visit **[keywrit.github.io/hub](https://keywrit.github.io/hub/)** to generate license tokens.

## Development

```bash
bun install
bun run dev
```
